## 下载angular
- 先去生成一个package.json
生成后 在安装时会以当前package.json文件位置来安装模块
```
npm init
```
- npm (管理后台模块)
下载node后才会有，node package manager
```
npm install angular 
```

>  使用npm install 安装时如果上一级有node_modules文件会安装到上一级上
> 文件名字不能是中文，而且不要以已经存在的项目名字命名

```
npm init -y //项目名称不能中文，不能大写
```
